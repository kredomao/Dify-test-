# Dify チャットボット - 公式サポートボット

このリポジトリには、Difyで作成した公式サポートチャットボットの設定ファイルとドキュメントが含まれています。

## 📁 ファイル構成

- `sample_support_document.md` - ナレッジベース用のサンプルサポートドキュメント
- `dify_chatbot_workflow.json` - Difyワークフロー設定ファイル（DSL）

## 🤖 チャットボットの仕様

- **モデル**: gpt-4o
- **機能**: ユーザーの質問に対して、ナレッジベースから情報を検索して回答
- **出力形式**: 【結論】【根拠】【出典】の3部構成
- **Retrieval設定**: top_k=5

## 🚀 使い方

### 1. Difyにインポート

1. [Dify](https://dify.ai/)にログイン
2. 「スタジオ」→「新しいアプリを作成」
3. 「チャットボット」を選択

### 2. ナレッジベースの設定

1. 「Knowledge」セクションでナレッジベースを作成
2. `sample_support_document.md`をアップロード
3. チャットボットにナレッジベースを追加

### 3. プロンプトの設定

「INSTRUCTIONS」に以下を入力：

```
あなたは〇〇会社の公式サポートボットです。以下のルールに従って回答してください：

1. 提供されたナレッジベースの情報のみを使用して回答してください
2. 回答は必ず以下の構成で出力してください：
   【結論】
   - 質問に対する簡潔な答え
   
   【根拠】
   - 結論を導いた理由や詳細説明
   - 関連する具体的な情報
   
   【出典】
   - 参照したドキュメント名やセクション

3. 情報が不足している場合は、正直にその旨を伝えてください
4. 丁寧で分かりやすい言葉遣いを心がけてください
```

## 📝 サンプルドキュメントの内容

`sample_support_document.md`には以下の情報が含まれています：

- 料金プラン（無料/スタンダード/プレミアム）
- 返金ポリシー
- アカウント登録方法
- パスワードリセット
- データバックアップ
- サポート対応時間
- 契約解約方法
- セキュリティ対策
- トラブルシューティング

## 🧪 テスト質問

チャットボットをテストする際は、以下の質問を試してください：

- 「返金ポリシーについて教えてください」
- 「料金プランの種類は？」
- 「プレミアムプランの特徴を教えて」
- 「解約方法を教えてください」

## 🔧 カスタマイズ

### 会社名の変更

プロンプト内の「〇〇会社」を実際の会社名に変更してください。

### ナレッジベースの追加

実際のサポートドキュメント、FAQ、マニュアルなどをナレッジベースに追加することで、より実用的なサポートボットになります。
