# 〇〇会社 サポートドキュメント

## 製品概要

〇〇会社の製品は、お客様のビジネスを効率化するためのクラウドベースのソリューションです。
本製品は2020年よりサービスを開始し、現在3000社以上のお客様にご利用いただいております。

## よくある質問

### Q1: 料金プランについて

**プラン種類:**
- **無料プラン**: 月額0円、基本機能のみ利用可能、ユーザー数3名まで
- **スタンダードプラン**: 月額5,000円、全機能利用可能、ユーザー数10名まで
- **プレミアムプラン**: 月額15,000円、全機能+優先サポート、ユーザー数無制限

**支払い方法:**
クレジットカード、銀行振込、請求書払いに対応しています。

### Q2: 返金ポリシー

**返金対応について:**
サービスにご満足いただけない場合、初回お申し込みから30日以内であれば全額返金いたします。

**返金手続き:**
1. サポート窓口（support@example.com）にご連絡ください
2. 返金理由を簡単にお知らせください
3. 5営業日以内に返金処理を行います
4. お客様の口座に返金されるまで、7-10営業日かかります

**注意事項:**
- 2回目以降のお申し込みは返金対象外です
- 年間契約の場合は月割り計算での返金となります

### Q3: アカウント登録方法

**新規登録手順:**
1. 公式サイト（https://example.com）にアクセス
2. 「無料で始める」ボタンをクリック
3. メールアドレスとパスワードを入力
4. 確認メールが届くのでリンクをクリック
5. 基本情報を入力して登録完了

**所要時間:** 約3分で登録完了します。

### Q4: パスワードを忘れた場合

**パスワードリセット手順:**
1. ログイン画面で「パスワードを忘れた方」をクリック
2. 登録メールアドレスを入力
3. パスワードリセット用のリンクが送られます
4. リンクから新しいパスワードを設定

**セキュリティ:**
パスワードリセットリンクの有効期限は24時間です。

### Q5: データのバックアップ

**自動バックアップ:**
- お客様のデータは1日1回自動的にバックアップされます
- 過去30日分のバックアップを保持しています
- データセンターは国内2拠点で冗長化されています

**手動バックアップ:**
プレミアムプランのお客様は、いつでも手動でデータをエクスポートできます。

### Q6: サポート対応時間

**対応時間:**
- 無料プラン: メールサポートのみ（平日10:00-17:00、返信まで最大48時間）
- スタンダードプラン: メール・チャットサポート（平日9:00-18:00、返信まで最大24時間）
- プレミアムプラン: メール・チャット・電話サポート（平日9:00-20:00、土日祝10:00-18:00、返信まで最大4時間）

**緊急時:**
システム障害などの緊急時は、24時間365日対応いたします。

### Q7: 契約の解約方法

**解約手続き:**
1. マイページにログイン
2. 「設定」→「プラン管理」を選択
3. 「解約する」ボタンをクリック
4. 解約理由を選択（任意）
5. 確認画面で「解約を確定する」をクリック

**解約のタイミング:**
- 月末までに解約手続きを完了すると、翌月から課金が停止されます
- 日割り計算は行いません
- データは解約後30日間保管されます

### Q8: セキュリティ対策

**データ保護:**
- すべての通信はSSL/TLS暗号化されています
- データベースは AES-256 で暗号化されています
- ISO27001、SOC2認証を取得しています
- 定期的なセキュリティ監査を実施しています

**アクセス制御:**
- 二要素認証（2FA）に対応しています
- IPアドレス制限が可能です（プレミアムプランのみ）

## トラブルシューティング

### ログインできない場合

**確認事項:**
1. メールアドレスとパスワードが正しいか確認
2. Caps Lockがオンになっていないか確認
3. ブラウザのキャッシュをクリア
4. 別のブラウザで試す

それでも解決しない場合は、パスワードリセットを行ってください。

### 動作が遅い場合

**対処方法:**
1. ブラウザを最新版に更新
2. 不要なタブを閉じる
3. キャッシュとCookieをクリア
4. インターネット接続を確認

推奨環境: Chrome最新版、Firefox最新版、Edge最新版
